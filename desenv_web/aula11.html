Exemplos

  1. Simplificando o código com jQuery:

    JavaScript puro:

    document.getElementById(“teste”).value = 5;

    Mesmo código em JQuery:

    $(“#teste”).val(5);

  2. Manipulação de eventos

    Ao passar o mouse sobre a caixa, ela muda de cor. Ao retirar o mouse de cima, ela retorna a cor inicial.

    .exemplo

    Código:

    $(function(){
	$("#jquery_exemplo2").mouseover(function(){
		$(this).css('background-color', '#dc143c');
	});
	$("#jquery_exemplo2").mouseleave(function(){
		$(this).css('background-color', '#6495ed');
	});
})

    ou:

    $(function(){
	$("#jquery_exemplo2").on({
		mouseover:function(){
			$(this).css('background-color', '#dc143c');
		},
		mouseleave:function(){
			$(this).css('background-color', '#6495ed');
		}
	});
})

  3. Exibindo/ocultando elementos

    .exemplo

    Código:

    //dentro do document.ready
$("#bt3_ocultar").click(function(){
	$(".exemplo").hide();
})
$("#bt3_aparecer").click(function(){
	$(".exemplo").show(200);
})
$("#bt3_revezar").click(function(){
	$(".exemplo").toggle(500);
})
