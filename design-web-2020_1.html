Aulas

Materiais de apoio

  • Aula1

    Introdução a Internet

    Discutindo a evolução da internet, tecnologias relacionadas, padrões web e protocolos mais usados.

  • Aula2

    Introdução ao HTML

    Linguagens de marcação e dados semi-estruturados; HTML - marcações e sintaxe básica. Aprenda sobre marcações de texto e listas.

  • Aula3

    HTML - Links, Imagens e Listas

    Aprenda as marcações básicas para exibição de imagens e links. Aprenda sobre endereçamento de recursos. Tipos de listas e estruturação de itens.

  • Aula4

    HTML - Tabelas e Frames

    Organize dados tabulares utilizando marcações para tabelas. O que são frames e como utilizá-los para mashups.

  • Aula5

    Arquitetura da Internet

    Entenda como está organizada a rede mundial de computadores, seus componentes e protocolos principais.

  • Aula6

    Formulários

    Como enviar dados para o servidor através de páginas HTML utilizando formulários.

  • Aula7

    HTML 5

    Aprenda sobre os novos recursos disponíveis na versão HTML5. O que é HTML semântico e qual a importância em utilizá-lo?

  • Aula8

    Cascade Style Sheet - CSS

    Introdução ao CSS - sintaxe e famílias de propriedades. Seletores de CSS. Bordas, margens e espaçamento interno de elementos.

  • Aula9

    CSS - Cores e Imagens

    Aprimore a apresentação de sites com cores e imagens de fundo.

  • Aula10

    CSS - Propriedades tipográficas e fontes

    Aprenda a manipular espaçamentos de fontes e linhas e estilos diversos.

  • Aula11

    Diagramação de sites com CSS

    Aprenda a esquematizar telas e objetos. Posicionamento, altura, largura e sobreposição de elementos.

  • Aula12

    CSS3

    Aprenda sobre as novidades trazidas pelo CSS3, como pseudo-classes e pseudo-elementos; efeitos de borda e sombra.

  • Aula13

    CSS3 - Cores e Fundos

    Aprenda a utilizar recursos de transparências e gradientes.

  • Aula14

    CSS3 - Animação

    Aprenda a utilizar recursos de transformações e transições.

  • Aula15

    Interface de dispositivos móveis

    Aprenda sobre viewport e os recursos de media query. Projeto de interfaces baseados em grids.

  • Aula16

    Frameworks de CSS e o Bootstrap

    Buscando componentes de terceiros para agilizar projetos.

  • Aula17

    User Experience - UX

    De que formas podemos conceber projetos de interfaces mais inclusivos e atraentes aos nossos clientes?

  • Aula18

    UX - Traçando um público-alvo

    Investigando o perfil do usuários mais recorrentes.

  • Aula19

    UX - Internet Inclusiva

    Recursos para tornar a Web mais inclusiva.

  • Aula20

    UX - Usabilidade

    Como avaliar a qualidade das interfaces produzidas?
